1. 队列是只用push和shift完成增删的数组, 只能允许从尾部添加元素, 只允许从头部移除元素

2. 下面基于队列, 完成"小册姐姐排队"和"小册姐姐取餐"的过程

```javascript
const queue = [];

queue.push('小册姐姐1');
queue.push('小册姐姐2');
queue.push('小册姐姐3');

while(queue.length) {
  const head = queue[0]
  console.log(`${head} 去取餐`);
  queue.shift();
}

console.log(queue)
```
